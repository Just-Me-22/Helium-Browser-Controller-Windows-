# Helium-Windows-Raycast

A browser control extension for the Windows version of Raycast. This project extends Raycast's functionality to Windows users, offering essential browser automation capabilities that were previously available only on macOS.

## Features

- Create new browser tabs
- - Search bookmarks
  - - Search browsing history
    - - Open private and standard browser windows
      - - Close all browser windows
